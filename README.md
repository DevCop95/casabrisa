# 🌊 Casa Brisa · Asistencia Turística para Cartagena

Sitio web de asistencia turística para visitantes de Cartagena. Itinerarios, reservas, traslados y acompañamiento remoto.

> *“Murallas, balcones, brisa del Caribe y apoyo real para tu viaje.”*

## ✨ Características

- Landing page completa con secciones de servicios
- Tipografías: Bricolage Grotesque + Instrument Serif
- Diseño responsive · HTML/CSS/JS puro · sin framework
- Desplegado en GitHub Pages
